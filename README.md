## Run the project

| Commands        | What does it do?                   | Observação                     |
| -------------   | -------------                | -----                          |
| `npm install`   | Install necessary packages   | **Run it first**               |
| `npm start`     | Start Dev environment        | Access `http://localhost:1234` |
| `npm run build` | Build the project            | Result in `dist` folder        |
